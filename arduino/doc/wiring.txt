Cellulaire
==========
La communication s'effectue par le port UART 1.

WiFi
====
La communication s'effectue via SoftwareSerial.

Arduino Pro
RX: Pin TX
TX: Pin RX
Initialisation: SoftwareSerial wifiSerial(RX, TX);

GPS
===
La communication s'effectue via SoftwareSerial.

Arduino Pro
RX: Pin 7
TX: Pin 6
Initialisation: SoftwareSerial gpsSerial(7, 6);

Accelerometre
=============
La communication s'effectue via I2C.

Console 
=======
La communication s'effectue via SoftwareSerial.

Pin 11 : Tx carte
Pin 12 : Rx carte
Initialisation: SoftwareSerial consoleSerial(12, 11)
