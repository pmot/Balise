Ce document décrit la connexion au MCU Arduino Pro

Cellulaire
==========
La communication s'effectue par le port UART 1.

WiFi
====
La communication s'effectue via SoftwareSerial.

Arduino Pro
RX: Pin TX
TX: Pin RX
Initialisation: SoftwareSerial wifiSerial(RX, TX);

GPS
===
La communication s'effectue via SoftwareSerial.

Arduino Pro
RX: Pin 7
TX: Pin 6
Initialisation: SoftwareSerial gpsSerial(7, 6);

Accelerometre
=============
La communication s'effectue via I2C.

LED 
===
pin 13

Console 
=======
La communication s'effectue via SoftwareSerial.

Pin 11 : Tx carte
Pin 12 : Rx carte
Initialisation: SoftwareSerial consoleSerial(12, 11)

Connecteur externe
===================
1 - Tx Console
2 - Rx Console
3 - Tx upgrade
4 - Rx upgrade
5 - Bat +
6 - Masse
7 - 3,3 v (sortie venant de la carte ) ... je sais pas pourquoi ....
8 - RTS

/!\ Les Tx et Rx sont vus coté arduino

Information sur les cable USB TTL 5V : http://info.ebconnections.com/
